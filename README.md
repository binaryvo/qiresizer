# qiresizer

##Getting started##

###Setup project dependencies:###

```
php >= 5.3,
Apache Server >= 2.4
Imagemagick php extension
composer Dependency Manager
```

###Clone the project###

```
git clone git@github.com:binaryvo/qiresizer.git
```


###Install composer and projects dependencies###


```
composer install --dev
```
